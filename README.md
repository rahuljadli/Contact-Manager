# Contact Manager

## Made a contact mangaging web app build completely on spring framweork 
### Used spring modules for front end and backend

### used SpringBoot for backend development for the creation and consuption of Rest API's.
### used Spring MVC for front end development ,provided secure access to the web app using spring security for authentication and authorization
### used Spring ORM for storing the users and contacts details in MYSQl DB.
